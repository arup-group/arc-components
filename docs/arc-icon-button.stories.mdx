# Icon Button
### `<arc-icon-button>`

The ArcIconButton is a simple, icon-only button that can be used for actions and in toolbars.
For a full list of icons that come bundled with Arc, refer to the #link input here.

## Examples
### Colors
Icon buttons are designed to have a uniform appearance, so their color is not inherited.
However, you can still customize them by styling the css variables.
<Canvas>
  <Story name="Default" story={stories.Default} />
  <Story name="CustomColor" story={stories.CustomColor} />
</Canvas>

### Link button
It's often helpful to have a button that works like a link.
This is possible by setting the `href` attribute, which will make the component render an `<a>` under the hood.
This gives all the default link behavior the browser provides (e.g. CMD/CTRL/SHIFT + CLICK) and exposes the target and download attributes.

  <Story name="Link" story={stories.Link} />

### States
Use the `active`, `disabled` or `loading` attributes to change the state of the button.
