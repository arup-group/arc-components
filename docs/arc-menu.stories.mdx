# Menu
### `<arc-menu>`

The Menu provides a list of options for the user to choose from.
You can use [arc-menu-items](?path=/story/arcmenuitem--default) to compose an interactive menu.
Menus support keyboard interactions, including type-to-select to select an option.

  <Story name="Default" story={stories.Default} />

## Events

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| arc-select   | Emitted when a menu item is selected                      |

## Methods

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| typeToSelect | Automatically selects an option based on what the user is currently typing.|

## Slots

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| (default)    | The menu's content, including menu items                  |
