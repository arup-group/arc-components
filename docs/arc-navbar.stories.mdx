# ArcNavbar
### `<arc-navbar>`

The `<arc-navbar>` component includes support for branding and navigation and provides out of the box responsiveness for all devices.
Use [arc-buttons](?path=/story/arcbutton--tab) or [arc-icon-buttons](?path=/story/arciconbutton--default) to compose the navbar.
