# `ArcTable`

**ARC** provides support for data visualization by providing an `ArcTable` wapper component around [Grid.js](https://gridjs.io/).

## Usage
``` html
<div id="table"></div>
```

```ts
import '@arc-web/components/components/table/arc-table.css';
import ArcTable from '@arc-web/components/components/table/ArcTable';

const table = documentation.querySelector('#table')!;
const arcTable = new ArcTable.Grid({
  columns: ["Name", "Email", "Phone Number"],
  data: [
    ["John", "john@example.com" , "(353) 01 222 3333"],
    ["Mark", "mark@exmaple.com", ""],
  ],
  resizable: true,
  search: true,
});
arcTable.render(table);
```

> **Note**: For more advanced features of the `ArcTable` component, check the official [GridJS](https://gridjs.io/docs/) docs.
