# Button
### `<arc-button>`

<Story name="Contained" story={stories.Contained} />

Buttons represent actions that are available to the user.

## Examples
### Types
Use the `type` attribute to set the button's type.

### Colors
Use the `color` attribute to set the button's color.

### Sizes
Use the `size` attribute to set the button's size

### Link button
It's often helpful to have a button that works like a link.
This is possible by setting the `href` attribute, which will make the component render an `<a>` under the hood.
This gives all the default link behavior the browser provides (e.g. CMD/CTRL/SHIFT + CLICK) and exposes the target and download attributes.

### States
Use the `active` attribute to activate a button.

Use the `loading` attribute to make a button busy.
Clicks will be suppressed until the loading state is removed.

Use the `disabled` attribute to disable a button.
Clicks will be suppressed until the disabled state is removed.

## Slots

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| (default)    | The button's label.                                       |
| prefix       | Used to prepend an icon or similar element to the button. |
| suffix       | Used to append an icon or similar element to the button.  |

## CSS Parts

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| base         | The component's base wrapper.                             |
| prefix       | The prefix container.                                     |
| label        | The button's label.                                       |
| suffix       | The suffix container.                                     |

## CSS Variables

| Name             | Description                                           |
|------------------|-------------------------------------------------------|
| width            | Set the width of the button.                          |
| --min-width      | Set the min width of the button.                      |
| --btn-color      | Overwrite the font color of the button.               |
| --btn-background | Overwrite the background color of the button          |
