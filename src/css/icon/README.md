# ARC CSS: `icon`

### Default

```html
<svg class="arc-icon">
  <use href="/assets/icons.svg#arc-{ICON_NAME}" />
</svg>
```

### Size

```html
<svg class="arc-icon" style="--arc-icon--size: var(--arc-font-size-medium)">
  <use href="/assets/icons.svg#arc-{ICON_NAME}" />
</svg>
```

### Color

```html
<svg class="arc-icon" style="--arc-icon--color: rgb(var(--arc-blue-050));">
  <use href="/assets/icons.svg#arc-{ICON_NAME}" />
</svg>
```
