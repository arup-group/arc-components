# Accessibility
### `<arc-accessibility>`

The ArcAccessibility allows users to change certain aspects of the website or application to fit their needs.
They can change settings like:
* Theme
* FontSize
* LineHeight
* LetterSpacing
* etc.

By default, the [arc-container](?path=/story/arccontainer--default) component imports the accessibility panel
and handles all the above out of the box. When used without the [arc-container](?path=/story/arccontainer--default) component,
your application can listen to the `arc-accessibility-change` event fired from the accessibility component.

```bash
<arc-accessibility id='accessibility'></arc-accessibility>
<script>
  document.getElementById('accessibility').addEventListener('arc-accessibility-change', e => {
    const { preferences } = e.detail;
    console.log(preferences);
  })
</script>
```

This event throws out the user defined values like so:
```bash
 {
  fontSize: "medium"
  highLegibilityFonts: false
  highlightLinks: false
  letterSpacing: "normal"
  lineHeight: "normal"
  plainText: false
  theme: "dark"
 }
```

## Events

| Name                        | Description                                |
|-----------------------------|--------------------------------------------|
| arc-accessibility-change    | Emitted when the user preferences change   |

## Methods

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| show         | Shows the accessibility drawer.                           |
| hide         | Hides the accessibility drawer.                           |
