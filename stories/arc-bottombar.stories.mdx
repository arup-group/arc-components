# ArcBottombar
### `<arc-bottombar>`

The `<arc-bottombar>` component includes support for navigation and provides out of the box responsiveness for all devices.
Use [arc-icon-buttons](?path=/story/arciconbutton--default) to compose the bottom bar.
On mobile devices, this component will be displayed within the [arc-container](?path=/story/arccontainer--container) component automatically,
to ensure a functional and layout-friendly application at any time.

### When to use:
* Top-level destinations that need to be accessible from anywhere in the app
* Three to five destinations
* Mobile or tablet only

### When not to use:
* Single tasks, such as viewing a single email
* User preferences or settings
