### Link button
It's often helpful to have a button that works like a link.
This is possible by setting the `href` attribute, which will make the component render an `<a>` under the hood.
This gives all the default link behavior the browser provides (e.g. CMD/CTRL/SHIFT + CLICK) and exposes the target and download attributes.

## Methods

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| click        | 	Simulates a click on the button.                         |

## Slots

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| (default)    | The button's label.                                       |
| prefix       | Used to prepend an icon or similar element to the button. |
| suffix       | Used to append an icon or similar element to the button.  |

## CSS Variables

| Name             | Description                                           |
|------------------|-------------------------------------------------------|
| width            | Set the width of the button.                          |
| --min-width      | Set the min width of the button.                      |
| --btn-color      | Overwrite the font color of the button.               |
| --btn-background | Overwrite the background color of the button          |
