# Card
### `<arc-card>`

The Arc card is used display summarized content and its respective actions. It contains basic slots for the user to place a heading, an optional image background and [arc-buttons](?path=/story/arc-button--default) 
The Arc card is designed to be accessible with a property for alt text if the image is used.

## Slots

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| heading         | This slot is used to add a heading to the carc.              |
| content         | This slot is used to add the main content to the card.     |
| actions         | This slot should be used to add footer items like buttons.   |
