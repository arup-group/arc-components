# ArcHero

### `<arc-hero>`

The ArcHero is used to create custom hero component.
By default, it contains the title on the left hand side, the subtitle/text/image on the right hand side.
On mobile devices, title and subtitle/text/image are located as in one column.

<Canvas>
  <Story name="Default" story={stories.Default} />
</Canvas>

## Examples

### Types

Use `bgImg` type to set background image to the hero component

```bash
<html>
  ...
  <body>
    <arc-hero bgImg="https://random.imagecdn.app/500/150">
    ...
    </arc-hero>
  </body>
</html>
```

which results in the following
