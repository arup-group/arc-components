# ArcNavbar
### `<arc-navbar>`

The ArcNavbar includes support for branding and navigation and provides out of the box responsiveness for all devices.
Use [arc-buttons](?path=/story/arcbutton--tab) or [arc-icon-buttons](?path=/story/arciconbutton--default) to compose the navbar.

## Slots

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| (default)    | This slot is used to add tabs to the navbar.              |
| name         | This slot is used to add a tool name or sub branding.     |
| user         | This slot should be used to display the signed in user.   |
