# ArcRadio
### `<arc-radio>`

The ArcRadio is used to select one option from a group of many.

Radios are designed to be used with [arc-radio-groups](?path=/story/arcradiogroup--default).
As such, all the examples on this page utilize them to demonstrate their correct usage.

## Events

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| arc-change   | Emitted when the control's checked state changes.         |

## Methods

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| click        | Simulates a click on the radio.                           |
| focus        | Sets focus on the radio.                                  |
| blur         | Removes focus from the radio.                             |

## Slots

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| (default)    | The radio's label.                                        |
