## Methods

| Name            | Description                                            |
|-----------------|--------------------------------------------------------|
| login           | Attempts to login through Azure AD                     |
| logout          | Attempts to logout from Azure AD                       |
| getAccount      | Retrieve the Azure AD account                          |
| isAuthenticated | Check whether the user is authenticated                |

## Slots

| Name         | Description                                               |
|--------------|-----------------------------------------------------------|
| logout       | Used to overwrite the way a user is being logged out.     |
| login        | Used to overwrite the way a user is being logged in.      |
